Protection and Rolling Counter Configuration

The Protection and Rolling Counter custom device allows the user to easily select data channels for calculating a CRC or Checksum. It creates and maps rolling counters and protection channels to selected outgoing signals.

A quick start document provides an overview of the addon capabilities. It is located at:

"C:\Users\Public\Documents\National Instruments\NI VeriStand (Year)\Custom Devices\Protection and Counter Addon\Windows\Quick Start Documentation\Protection and Counter Addon Quick Start.html"

Most of the addon sections contain additional built-in documentation.

What do you want to do?

Change CAN Settings

The default CAN settings are imported from the database, and can be changed when first adding the custom device to the VeriStand system definition. You can also change the CAN port settings in the CAN Settings section of this page. Note: Check the Legacy? checkbox for VeriStand 2017 (and earlier) implementations of CAN FD. CAN frame generation is handled in an Asynchronous loop with a priority set by the Loop Priority setting. Please see the LabVIEW Help for additional information on applicable values.

Add A Frame Configuration

To add a frame configuration, right-click on the 'Cyclic' message type in the configuration tree, and select Import Frames.

Disable Add-on

To disable this add-on, right-click on the add-on instance and select Disable from the shortcut menu.

Delete multiple items

To delete multiple items, right-click the object in the configuration tree and select Delete Multiple Items from the shortcut menu.
